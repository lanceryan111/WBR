Maccy 作为一个极简主义的剪贴板工具，最爽的地方就是全键盘操作。其实它**默认就已经自带了一个非常好用的快捷键**，你可以直接用，也可以随时修改它。
## ⌨️ 默认快捷键（即开即用）
你刚装好 Maccy 之后，不需要任何设置，直接在键盘上按下：
> **Shift (⇧) + Command (⌘) + C**
> 
Maccy 的历史记录列表就会**精准弹出在你当前光标所在的位置**。接着直接打字就能搜索历史，按回车（Enter）就能直接粘贴。
## ⚙️ 怎么修改成自己喜欢的快捷键？
如果你想把它改成和 Alfred 剪贴板一样，或者换成更顺手的键，请按以下步骤操作：
### 1. 打开 Maccy 的设置面板
 * 方法 A：按下默认快捷键 Shift + Command + C 唤出面板，然后直接按下 Mac 通用的设置快捷键 **Command (⌘) + ,（逗号）**。
 * 方法 B：在 Mac 屏幕右上角的菜单栏，找到 Maccy 的图标（通常是一个小剪贴板或者单行列表的图标），点击它，在下拉菜单中选择 **Preferences...**（或 **Settings...**）。
### 2. 修改快捷键
 1. 在弹出的设置窗口中，最顶部第一项就是 **Hotkey**（快捷键）。
 2. 用鼠标点击一下那个快捷键输入框。
 3. 直接在你的键盘上**按下你想设定的全新组合键**（例如：Option + Command + C）。
 4. 录入成功后，直接关闭窗口即可，新快捷键立即生效！
## 🚀 附赠：Maccy 核心高效操作流
既然用上了 Maccy，这几个全键盘高阶操作能让你找命令的速度飞起来：
 * **快速选定：** 唤出历史后，列表右侧会自带数字编号（1、2、3...）。你不需要用方向键上下选，直接按 **Command (⌘) + 数字键** 就能秒切对应的历史记录。
 * **直接粘贴 / 纯文本粘贴：** * 默认情况下，按 Enter 是复制该项。
   * 如果在设置里勾选了 **Paste Automatically**（自动粘贴），按 Enter 就会直接贴进文本框。
   * 按住 **Option (⌥) + Enter** 可以实现“格式化擦除”，只粘贴纯文本。
 * **固定常用条目（Pin）：** 经常要用的固定命令或代码，选中它按 **Option (⌥) + P**，这条记录就会被死死钉在最顶部，永远不会被新复制的内容冲掉。
